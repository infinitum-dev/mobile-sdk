getAllApis

fun getAllApis(onSuccess: (List<ApiResponse>) -> Unit, onFailure: (ErrorResponse) -> Unit): Unit

Gets all the existing Apis. Invokes onSuccess if the request was successful, onFailure otherwise. The onSuccess returns a list of ApiResponse that contains all Apis. The onFailure returns an ErrorResponse that contains information about what went wrong.
