authenticate

fun authenticate(email: String, password: String, onSuccess: (AuthResponse) -> Unit, onFailure: (ErrorResponse) -> Unit): Unit

Authenticates the user by his email with the password. Invokes onSuccess if the request was successful, onFailure otherwise. The onSuccess returns an AuthResponse that contains data about the User and a token to be used by the SDK. The onFailure returns an ErrorResponse that contains information about what went wrong.
