config

fun config(domain: String, appType: String, onSuccess: (ConfigResponse) -> Unit, onFailure: (ErrorResponse) -> Unit): Unit

Invokes onSuccess if the request to the given domain for applications with the appType was successful. Invokes onFailure otherwise. Returns a ConfigResponse object that contains a list of applications when successful or an ErrorResponse if the request failed.
