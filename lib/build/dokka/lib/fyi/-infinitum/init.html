init

fun init(domain: String, appToken: String, onSuccess: (InitResponse) -> Unit, onFailure: (ErrorResponse) -> Unit, eventBuilder: NodeEvent.NodeEventBuilder): Unit

Invokes onSuccess if the request to initialize the Application with the given appToken was successful. Invokes onFailure otherwise. Use eventBuilder to tell the SDK which Socket events need to be listened to.
