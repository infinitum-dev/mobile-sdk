getDevicePositionsByDeviceId

fun getDevicePositionsByDeviceId(deviceId: Int, onSuccess: (List<DevicePositionResponse>) -> Unit, onFailure: (ErrorResponse) -> Unit): Unit

Gets a list of DevicePositions by the associated deviceId. Invokes onSuccess if the request was successful, onFailure otherwise. The onSuccess returns a list of DevicePositionResponse associated to that Device. The onFailure returns an ErrorResponse that contains information about what went wrong.
