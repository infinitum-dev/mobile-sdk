getAllDevices

fun getAllDevices(onSuccess: (List<DeviceResponse>) -> Unit, onFailure: (ErrorResponse) -> Unit): Unit

Gets all Devices of the Infinitum instance. Invokes onSuccess if the request was successful, onFailure otherwise. The onSuccess returns a DeviceResponse list. The onFailure returns a ErrorResponse that contains information about what went wrong.
