validateDeviceLicensed

fun validateDeviceLicensed(deviceIdentity: String, onSuccess: (DeviceResponse) -> Unit, onFailure: (ErrorResponse) -> Unit): Unit

Validates if the Device with the identity deviceIdentity is Licensed. Invokes onSuccess if the request was successful, onFailure otherwise. The onSuccess returns the DeviceResponse associated with the deviceIdentity. The onFailure returns a ErrorResponse that contains information about what went wrong.
