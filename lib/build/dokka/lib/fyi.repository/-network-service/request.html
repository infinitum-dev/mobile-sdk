suspend fun request(url: String, headerParameters: MutableMap<String, String>?, bodyParameters: MutableMap<String, String>?, httpMethod: <ERROR CLASS>): Any?

Single function that will handle all types of requests.

Makes an HttpRequest to the url of the type httpMethod where the header of the request is made with the headerParameters and the body of the request with the bodyParameters.

Return
ErrorResponse object if an error has occurred or a String which will be the body of the HTTP response.
