getAllDeviceInputs

fun getAllDeviceInputs(onSuccess: (List<DeviceInputResponse>) -> Unit, onFailure: (ErrorResponse) -> Unit): Unit

Gets the list of all DeviceInputs. Invokes onSuccess if the request was successful, onFailure otherwise. The onSuccess returns a List of DeviceInputResponse that contains all the DeviceInputs. The onFailure returns an ErrorResponse that contains information about what went wrong.
