getDeviceInputsByDeviceTypeId

fun getDeviceInputsByDeviceTypeId(deviceTypeId: Int, onSuccess: (List<DeviceInputResponse>) -> Unit, onFailure: (ErrorResponse) -> Unit): Unit

Gets a list of DeviceInputs related to a DeviceType by deviceTypeId. Invokes onSuccess if the request was successful, onFailure otherwise. The onSuccess returns a List of DeviceInputResponse that represent all the inputs of that device type. The onFailure returns an ErrorResponse that contains information about what went wrong.
