getRequestsErrorsCountBetweenTwoDates

fun getRequestsErrorsCountBetweenTwoDates(builder: RequestsOptionalParameters.Builder, onSuccess: (RequestCountErrors) -> Unit, onFailure: (ErrorResponse) -> Unit): Unit

Gets the request count by errors between the dates in the builder. Invokes onSuccess if the request was successful, onFailure otherwise. The onSuccess returns a list of RequestCountErrors. The onFailure returns an ErrorResponse that contains information about what went wrong.
