getRequestsByModuleBetweenTwoDates

fun getRequestsByModuleBetweenTwoDates(builder: RequestsOptionalParameters.Builder, onSuccess: (Map<String, List<RequestResponse>>) -> Unit, onFailure: (ErrorResponse) -> Unit): Unit

Gets all existing requests by module between the dates in the builder. Invokes onSuccess if the request was successful, onFailure otherwise. The onSuccess returns a map where the Key is the module name and the value is a list of RequestResponse. The onFailure returns an ErrorResponse that contains information about what went wrong.
